## What It Does

This utility automates target acquisition inside the Dig Deeper game environment on Brainrots. It is aimed at helping players move more efficiently through resource gathering and combat by adding consistent aimbot-style assistance that works through the game interface. The current version emphasizes sharper targeting behavior while staying aligned with newer game updates.

The script is intended for PC use and runs as a lightweight executor rather than a permanent install. Its design keeps the workflow close to the game's native mechanics, so users can spend less time adjusting aim by hand and more time making decisions in play. Ongoing releases are used to keep the script working as the Brainrots platform changes.
